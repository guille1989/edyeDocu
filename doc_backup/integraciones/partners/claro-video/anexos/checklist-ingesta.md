# Checklist de Ingesta – Claro Video

Checklist operativo para validar una ingesta completa y correcta para
el partner **Claro Video**.

---

## Pre-ingesta

- [ ] Contenido cargado en JW Player
- [ ] Playlists correctas
- [ ] Metadata completa
- [ ] Posters y stills disponibles
- [ ] Naming validado

---

## Ingesta

- [ ] JW Player sincronizado con EDYE API
- [ ] Delivery creado para Claro Video
- [ ] Validaciones aprobadas
- [ ] Assets en estado Processing / Completed

---

## Post-ingesta

- [ ] Assets entregados vía Aspera / SFTP
- [ ] Validación final completada
- [ ] Reporte descargado
- [ ] Partner notificado (si aplica)
