## Checklist Compacto — Ingesta Claro Brasil

| Fase         | Validación clave     | Criterio mínimo                                  | Responsable |
| ------------ | -------------------- | ------------------------------------------------ | ----------- |
| Pre-ingesta  | Video listo          | H.264 · ≥720p · ≤2h                              | Content Ops |
| Pre-ingesta  | Metadata obligatoria | IDs externos (TMS/Gracenote) + campos requeridos | Content Ops |
| Pre-ingesta  | Packaging requerido  | Cumple spec Claro (naming/estructura)            | Content Ops |
| Pre-ingesta  | Artes solicitadas    | Posters / stills según spec                      | Content Ops |
| Diseño       | Artes creadas        | Ratios/tamaños Claro                             | Design Team |
| Diseño       | Artes cargadas       | Imágenes en EDYE                                 | Design Team |
| Ingesta      | Sync JWP → EDYE      | Contenido sincronizado                           | Content Ops |
| Ingesta      | Delivery generado    | Media + metadata + imágenes                      | DevOps      |
| Validación   | QC interno           | Video · metadata · imágenes OK                   | DevOps      |
| Entrega      | Canal acordado       | SFTP / Aspera (según Claro)                      | DevOps      |
| Entrega      | Transferencia OK     | Entrega completa sin errores                     | DevOps      |
| Post-ingesta | Validación final     | Aprobación Claro                                 | DevOps      |
| Cierre       | Cierre operativo     | Caso documentado                                 | Content Ops |
